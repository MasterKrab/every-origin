Every origin

Free CORS proxy for developers

How to use

Add to the url like param

{{ appUrl }}?url=https://example.com

Or add the target-url header

{ "target-url": "https://example.com" }

Examples

          
fetch("{{ appUrl }}?url=https://jsonplaceholder.typicode.com/posts")
          
fetch('{{ appUrl }}', {
  method: 'POST',
  headers: {
    'target-url': 'https://jsonplaceholder.typicode.com/posts',
  },
  body: { title: 'foo', body: 'bar', userId: 1 },
})
